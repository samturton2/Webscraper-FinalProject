FAQ

What is our project about?

The project is dedicated to creating an end-to-end automated pipeline of a Python webscraper app. The app is designed to retrieve data from ItJobsWatch, return it as a CSV file and show it as a list using HTML and Python Flask.


What is DevOps?

DevOps focuses on cross-disciplinary practices involving the development, deployment and maintenance of a service. As the name suggests, it is the combination of the development and operations team into one. This single team would then be responsible for the entire lifecycle of a product. Read more here!


What are the main aims in DevOps?

The main aim for those working in DevOps is to ensure the quick and efficient delivery of good code. To learn more about the tools and technologies commonly used within DevOps, check out our ToolChain page that explains what we used in this project. Read more here!
